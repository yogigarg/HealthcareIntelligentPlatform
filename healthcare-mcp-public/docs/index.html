Getting Started

Install with Cline

The easiest way to use this MCP server is through the Cline Marketplace:

  1. Open Cline
  2. Click the Extensions button in the toolbar
  3. Search for "Healthcare MCP"
  4. Click Install

Manual Installation

For manual installation or development:

git clone https://github.com/Cicatriiz/healthcare-mcp.git
cd healthcare-mcp
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
python run.py

Sample Usage in Cline

Once installed, you can access medical information directly in Cline conversations:

You: What are the side effects of metformin?

Cline: According to the FDA database, common side effects of metformin include:

  • Nausea and vomiting
  • Diarrhea
  • Abdominal discomfort
  • Decreased vitamin B12 levels

In rare cases, metformin can cause lactic acidosis, a serious condition that requires immediate medical attention.

API Access

For developers, the MCP server exposes a FastAPI interface with interactive documentation:

# Start the HTTP server
python run.py --http

# Access the API docs at:
# http://localhost:8000/docs

The server supports both stdio transport (for Cline) and HTTP/SSE transport (for web clients).
